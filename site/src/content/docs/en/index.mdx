`hikcamera-rs` is a Windows-only Rust workspace that wraps the HikCamera MVS
industrial camera C SDK. It exposes a small, typed surface that covers device
enumeration, camera configuration, streaming, image/video writers, and a
unified error model — while keeping `unsafe` strictly inside FFI boundaries.

## Two crates

<CardGrid>
  <Card title="hikcamera-sys" icon="seti:rust">
    Raw `bindgen` bindings generated from the vendor headers under
    `crates/hikcamera-sys/include/`. Re-exported as `hikcamera::sys`.
  </Card>
  <Card title="hikcamera" icon="seti:rust">
    Safe high-level wrapper. Public API lives in `crates/hikcamera/src/` and
    covers system lifecycle, device enumeration, camera control, streaming,
    image and video writers, and a unified error model.
  </Card>
</CardGrid>
