import { Tabs, TabItem } from '@astrojs/starlight/components';

`DeviceInfo` is the normalized, transport-agnostic view of an enumerated
device. Every field is parsed eagerly from the raw `MV_CC_DEVICE_INFO` so
downstream code does not need to know which transport produced it.

## Top-level fields

| Field | Type | Meaning |
| --- | --- | --- |
| `major_version` | `u16` | SDK device-info struct major version |
| `minor_version` | `u16` | SDK device-info struct minor version |
| `transport` | `Transport` | Transport-layer classification |
| `device_type` | `u32` | Raw product-type code from the SDK |
| `model` | `String` | Device model name |
| `serial` | `String` | Device serial number |
| `user_name` | `String` | User-configured name |
| `vendor` | `String` | Manufacturer name |
| `version` | `String` | Device version string |
| `family` | `String` | Product family name |
| `device_id` | `Option<u32>` | USB / CameraLink / GenTL device ID |
| `interface_id` | `Option<u32>` | Frame-grabber / interface ID |
| `mac` | `Option<String>` | Device MAC address |

## Transport-specific fields

<Tabs>
  <TabItem label="GigE">
    | Field | Meaning |
    | --- | --- |
    | `ip` | Current IP address |
    | `subnet` | Current subnet mask |
    | `gateway` | Current default gateway |
    | `net_export` | NIC IP the device is reachable through |
    | `host_ip` | Host IP currently holding the device |
    | `multicast_ip` | Multicast group IP |
    | `multicast_port` | Multicast group port |
    | `ip_config` | `IpConfig` (capability + current mode) |
  </TabItem>
  <TabItem label="USB">
    | Field | Meaning |
    | --- | --- |
    | `usb` | `UsbInfo` (USB endpoints + identifiers) |
  </TabItem>
  <TabItem label="Other transports">
    CameraLink, 1394, GenTL variants, and virtual device types populate the
    top-level fields but currently do not have dedicated transport-specific
    structs. Their raw bytes stay available through `MV_CC_DEVICE_INFO` if
    you drop down to `hikcamera::sys`.
  </TabItem>
</Tabs>

## `Transport` enum

| Variant | C macro | Notes |
| --- | --- | --- |
| `GigE` | `MV_GIGE_DEVICE` | GigE Vision |
| `Usb` | `MV_USB_DEVICE` | USB3 Vision |
| `CameraLink` | `MV_CAMERALINK_DEVICE` | CameraLink |
| `Ieee1394` | `MV_1394_DEVICE` | 1394-a/b |
| `VirtualGigE` | `MV_VIR_GIGE_DEVICE` | Virtual GigE Vision |
| `VirtualUsb` | `MV_VIR_USB_DEVICE` | Virtual USB3 Vision |
| `GenTlGigE` | `MV_GENTL_GIGE_DEVICE` | GenTL GigE |
| `GenTlCameraLink` | `MV_GENTL_CAMERALINK_DEVICE` | GenTL CameraLink |
| `GenTlCoaXPress` | `MV_GENTL_CXP_DEVICE` | GenTL CoaXPress |
| `GenTlXoF` | `MV_GENTL_XOF_DEVICE` | GenTL XoF |
| `GenTlVirtual` | `MV_GENTL_VIR_DEVICE` | GenTL virtual |
| `Other(u32)` | — | Unwrapped transport code |
| `Unknown` | `MV_UNKNOW_DEVICE` | SDK reported an unknown type |
