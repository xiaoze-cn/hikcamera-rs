`CameraParams.h` is the data-shape header of the HikCamera MVS C SDK. It
declares no functions — only the structs, enums, and `#define` constants
used by the entrypoints in `MvCameraControl.h`. The Rust wrapper re-exports
the bindgen-generated versions of these through `hikcamera::sys::*`.
