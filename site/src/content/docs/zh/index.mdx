`hikcamera-rs` 是一个 Windows-only 的 Rust workspace，封装海康 MVS 工业相机
C SDK。它把设备枚举、相机配置、采集、图像/视频写入和统一错误模型收敛到一个
小而类型化的 API 表面，同时把 `unsafe` 严格限制在 FFI 边界内。

## 两个 crate

<CardGrid>
  <Card title="hikcamera-sys" icon="seti:rust">
    通过 `bindgen` 从 `crates/hikcamera-sys/include/` 下的厂商头文件生成的
    原始绑定，作为 `hikcamera::sys` 重新导出。
  </Card>
  <Card title="hikcamera" icon="seti:rust">
    高层安全封装。公共 API 位于 `crates/hikcamera/src/`，覆盖系统生命周期、
    设备枚举、相机控制、采集、图像与视频写入以及统一错误模型。
  </Card>
</CardGrid>
