`DeviceInfo` 是已枚举设备的归一化、传输层无关视图。每个字段都从原始
`MV_CC_DEVICE_INFO` 即时解析，下游不需要知道是哪个传输层产生的。

## 顶层字段

| 字段 | 类型 | 含义 |
| --- | --- | --- |
| `major_version` | `u16` | SDK 设备信息结构主版本 |
| `minor_version` | `u16` | SDK 设备信息结构次版本 |
| `transport` | `Transport` | 传输层分类 |
| `device_type` | `u32` | SDK 返回的原始产品类型码 |
| `model` | `String` | 设备型号 |
| `serial` | `String` | 设备序列号 |
| `user_name` | `String` | 用户自定义名称 |
| `vendor` | `String` | 厂商 |
| `version` | `String` | 设备版本字符串 |
| `family` | `String` | 产品家族名 |
| `device_id` | `Option<u32>` | USB / CameraLink / GenTL 设备 ID |
| `interface_id` | `Option<u32>` | 采集卡 / 接口 ID |
| `mac` | `Option<String>` | 设备 MAC 地址 |

## 传输层专属字段

<Tabs>
  <TabItem label="GigE">
    | 字段 | 含义 |
    | --- | --- |

    | `subnet` | 当前子网掩码 |
    | `gateway` | 当前默认网关 |
    | `net_export` | 设备可达的网卡 IP |
    | `host_ip` | 当前占用设备的主机 IP |
    | `multicast_ip` | 组播组 IP |
    | `multicast_port` | 组播组端口 |
    | `ip_config` | `IpConfig`（能力 + 当前模式） |
  </TabItem>
  <TabItem label="USB">
    | 字段 | 含义 |
    | --- | --- |
    | `usb` | `UsbInfo`（USB 端点 + 标识） |
  </TabItem>
  <TabItem label="其它传输层">

## `Transport` 枚举

| 变体 | C 宏 | 说明 |
| --- | --- | --- |
| `GigE` | `MV_GIGE_DEVICE` | GigE Vision |
| `Usb` | `MV_USB_DEVICE` | USB3 Vision |
| `CameraLink` | `MV_CAMERALINK_DEVICE` | CameraLink |
| `Ieee1394` | `MV_1394_DEVICE` | 1394-a/b |
| `VirtualGigE` | `MV_VIR_GIGE_DEVICE` | 虚拟 GigE Vision |
| `VirtualUsb` | `MV_VIR_USB_DEVICE` | 虚拟 USB3 Vision |
| `GenTlGigE` | `MV_GENTL_GIGE_DEVICE` | GenTL GigE |
| `GenTlCameraLink` | `MV_GENTL_CAMERALINK_DEVICE` | GenTL CameraLink |
| `GenTlCoaXPress` | `MV_GENTL_CXP_DEVICE` | GenTL CoaXPress |
| `GenTlXoF` | `MV_GENTL_XOF_DEVICE` | GenTL XoF |
| `GenTlVirtual` | `MV_GENTL_VIR_DEVICE` | GenTL 虚拟 |
| `Other(u32)` | — | 未单独封装的传输层码 |
| `Unknown` | `MV_UNKNOW_DEVICE` | SDK 报告的未知类型 |
